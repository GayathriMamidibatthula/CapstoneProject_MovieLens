Capstone Project for HarvardX PH125.9x Data Science Course : MovieLens Project: A simple recommendation system using R on the movielens dataset

This capstone project is 1 out of 2 for the final course in the Harvard University Professional Certificate in Data Science. 

The main aim of this project is to create a movie recommendation system using the MovieLens dataset. The gist of the project includes training a machine learning algorithm using the inputs in the training set to predict movie ratings in the validation set.
